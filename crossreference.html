6.1 Internal cross references

Quarto website projects render all files separately. An important consequence it that it makes internal cross references between files not possible.

6.1.1 Default internal references

type display
level 1 Chapter 6
level 2 Section 6.1
level 3 Section 7.0.1
level 4 Section 7.0.1.1
figure Figure 7.1
table Table 8.1
equation Equation 9.1

6.1.2 Custom internal references

You can display only the number of the internal reference or replace the custom name with a custom name.

type only number custom
level 1 6 custom 6
level 2 6.1 custom 6.1
level 3 7.0.1 custom 7.0.1
level 4 7.0.1.1 custom 7.0.1.1
figure 7.1 custom 7.1
table 8.1 custom 8.1
equation 9.1 custom 9.1

6.2 External references

Don’t use URLs as is (like http://www.inbo.be or mailto:nobody@inbo.be).

Instead use the explicit markdown syntax like http://www.inbo.be, Google or e-mail
